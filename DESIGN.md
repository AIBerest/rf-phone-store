# Nordic Yellow

Source: https://designmd.ai/chef/nordic-yellow
License: MIT

Nordic Yellow is a Scandinavian-inspired e-commerce design system rooted in function, simplicity, democratic design, and a recognizable blue-and-yellow palette. The storefront should feel practical, warm, organized, and easy to scan.

## Color Palette

Primary: `#003399` for navigation, headers, primary actions, and brand identity.
Primary hover: `#002B80`.
Secondary: `#FFDA1A` for CTAs, price tags, and promotional accents. Do not use yellow text on white.
Neutral: `#767676`.
Background: `#F5F5F5`.
Surface: `#FFFFFF`.
Text primary: `#111111`.
Text secondary: `#484848`.
Border: `#DFDFDF`.
Success: `#0A8A00`.
Warning: `#E87400`.
Error: `#CC0008`.

## Typography

Display and body use `Noto Sans`.
Measurements and compact technical specs use `JetBrains Mono`.

Type scale:
- Display: 40px / 700
- H1: 32px / 700
- H2: 24px / 700
- H3: 18px / 700
- Body: 16px / 400
- Product name: 16px / 700
- Price: 24px / 700
- Dimensions/specs: 14px / 400, JetBrains Mono

## Components

Buttons are 44px high with 20px horizontal padding, 4px radius, and 15px/700 text.
Primary buttons use blue with white text. Yellow CTAs use yellow with near-black text.

Product cards use white background, 1px border, 4px radius, subtle elevation, 1:1 product imagery, 16px content padding, prominent prices, and compact spec chips.

Inputs use 44px height, 14px horizontal padding, 4px radius, 1px border, and blue focus state.

## Spacing

Base unit: 4px.
Scale: 4, 8, 12, 16, 20, 24, 32, 40, 48, 64, 80.
Container max width: 1400px.
Desktop product grid gap: 20px.
Mobile product grid gap: 8px.

## Rules

Use blue as the dominant UI color and yellow for promotion and action emphasis.
Keep layouts simple, grid-based, and functional.
Avoid gradients, decorative patterns, heavy shadows, and oversized rounded corners.
Show prices clearly with the currency symbol.
Use product photography/renders as the visual focus.
