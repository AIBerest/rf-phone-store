# RF Phone Store

Static storefront prototype for selling popular smartphones in Russia.

## Run Locally

```bash
python3 -m http.server 4173
```

Open `http://localhost:4173`.

## Included

- Five main storefront routes with hash navigation
- Real product imagery stored locally in `assets/products/`
- Source manifest for product images
- VoiceBox design system implementation from DesignMD
- `rf-phones` Codex skill in `skills/rf-phones`
